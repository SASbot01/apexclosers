# FINANZAS — modelo y proyección (Apex for closers)

> Estimaciones modeladas (no contabilidad). Mercado: closers hispanos. Precio **Token x3 ≈ 79€/mes**.
> COGS afinado con el cálculo real (Recall $0,50/h + IA + infra). Actualizado 2026-06-02.

## 1. Unit economics (por seat activo / mes)

| Concepto | € |
|---|---|
| Precio (Token x3) | **79** |
| COGS (Recall 40h + IA + infra) | **~30** |
| **Margen bruto** | **~49 (62%)** |
| Comisión afiliado (30% del precio, sobre seat activo) | ~24 |
| **Contribución a BW (vía afiliado)** | **~25** |
| Contribución a BW (vía ads/directo) | ~49 |
| **Contribución blended (≈70% afiliado / 30% directo)** | **~30** |

- COGS optimizado (Haiku + ASR propio): ~€23 → margen ~71%.
- Procesamiento de pago (~3% ≈ €2,4) ya restado del lado contribución.

## 2. LTV / CAC / break-even

- **LTV (a BW)** = contribución €30 × vida media.
  - Churn 5% → vida 20m → **LTV €600**
  - Churn 8% → vida 12,5m → **LTV €375**
  - Churn 12% → vida 8,3m → **LTV €250**
- **CAC**: afiliado pagado desde ingresos cobrados → **CAC upfront ≈ €0**. Ads: ~€15-50. → **LTV/CAC 8-25x**.
- **Break-even**: fijos lean ~€5-7k/mes ÷ €30 ≈ **170-235 seats activos** (alcanzable en ~3-4 meses con la red).

## 4. Sensibilidad al churn (lo que mueve la aguja)

| Churn/mes | Vida media | LTV | Lectura |
|---|---|---|---|
| 5% | 20 m | €600 | máquina imparable |
| 8% | 12,5 m | €375 | muy sano |
| 12% | 8,3 m | €250 | rentable pero crecer cuesta (a 5.000 seats hacen falta ~600 altas/mes solo para no caer) |

**Retención = el KPI nº1.** Anclar el producto al día a día del closer + planes anuales + "pausar" en vez de cancelar.

## 5. Tamaño de mercado (estimación)

- **TAM** (closers + setters + equipos de venta de infoproductores/agencias hispanos): **~80.000-120.000 seats**.
- **SAM** (activos, mid/high ticket, alcanzables vía comunidades/academias): **~30.000-40.000**.
- **SOM** (24-36 meses, 3-12% del SAM): **~1.500-5.000 seats** → encaja con la rampa.
- Competencia (Fathom/Gong/Otter/Fireflies): **English-first, genéricos/enterprise**, sin nicho closer ni IA en español → hueco real.

## 6. Veredicto

Camino plausible a **~1,5-2M€ ARR en 12-18 meses** y **~4-5M€ en 24-36**, con opcionalidad de 8 cifras (All-in-One + LatAm + Marketing). **Capital-efficient** (afiliado = CAC diferido) y **break-even bajo** (~200 seats). Las 3 palancas que deciden todo: **retención · disciplina de COGS (token con cuota+overage) · fiabilidad del producto**.
